# 🧮 Scientific Calculator (Python)

A modern GUI-based Scientific Calculator built with **Python** and **CustomTkinter**. The calculator supports basic arithmetic operations along with scientific functions like trigonometry, logarithms, square roots, exponents, and more.

## ✨ Features

- Basic arithmetic operations
  - Addition (+)
  - Subtraction (-)
  - Multiplication (×)
  - Division (÷)

- Scientific Mode
  - Square (x²)
  - Cube (x³)
  - Square Root (√)
  - Reciprocal (1/x)
  - Power (^)
  - Sine (sin)
  - Cosine (cos)
  - Tangent (tan)
  - Logarithm (log)
  - Natural Log (ln)
  - π (Pi)
  - Euler's Number (e)

- Percentage calculations
- Decimal number support
- Backspace and Clear buttons
- Keyboard shortcuts
- Clean and modern CustomTkinter interface
- Error handling for invalid expressions

## 📦 Installation

1. Clone the repository

```bash
git clone https://github.com/muskan-g22/Calculator.git
```

2. Navigate to the project folder

```bash
cd Calculator
```

3. Install the required package

```bash
pip install customtkinter
```

4. Run the application

```bash
python calculator.py
```

## ⌨️ Keyboard Shortcuts

| Key | Action |
|------|--------|
| Enter | Calculate Result |
| Backspace | Delete Last Character |
| Esc | Clear Display |
| + - * / | Arithmetic Operations |
| Numbers | Input Digits |
